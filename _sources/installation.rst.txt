Installation
============

Prerequisites
-------------

- **Python** >= 3.10
- **Node.js** >= 18 (optional, for web application)
- `Poetry <https://python-poetry.org>`__ (optional, for dependency management)

Using pip
---------

1. Clone the repository:

   .. code-block:: bash

      git clone https://github.com/YarynaRachkevych1/python-logo.git
      cd python-logo

2. Create a virtual environment:

   .. code-block:: bash

      python -m venv venv
      source venv/bin/activate

3. Install the project with dependencies:

   .. code-block:: bash

      pip install .

Using Poetry
------------

1. Clone the repository (if not already done):

   .. code-block:: bash

      git clone https://github.com/YarynaRachkevych1/python-logo.git
      cd python-logo

2. Install the project with dependencies:

   .. code-block:: bash

      poetry install
